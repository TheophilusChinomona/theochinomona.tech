## Usage Guidelines

### When to Use Light Theme

- Primary website experience
- Content-heavy pages
- Forms and input fields
- Documentation

### When to Use Dark UI Elements

- Code blocks
- Cards and containers for visual separation
- Navigation bars (optional)
- Footer sections (optional)
- Hero sections with gradient overlays

### Accessibility Considerations

- Ensure minimum contrast ratio of 4.5:1 for text
- Use focus rings (`#A5B4FC`) for keyboard navigation
- Test color combinations for colorblind accessibility
- Maintain sufficient contrast between text and background colors

---

## Implementation Notes

- The hybrid approach allows for a light primary theme while using dark elements strategically for visual interest and component separation
- Indigo provides a professional, modern feel that works across both light and dark contexts
- Gradients should be used sparingly and primarily in hero sections or high-impact call-to-action areas
